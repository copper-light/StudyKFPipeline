# 목표 (Objective)
이 스킬의 주요 목적과 이 스킬을 통해 달성하고자 하는 바를 설명합니다.

# 조건 (When to use)
에이전트가 어떤 상황에서 또는 어떤 사용자의 요청에 대해 이 스킬을 활용해야 하는지 구체적으로 명시합니다.

# 지침 (Instructions)
에이전트가 이 스킬을 수행할 때 반드시 따라야 하는 세부적이고 단계적인 규칙들입니다.
1. **1단계**: 해야 할 일을 명확하게 기재합니다.
2. **2단계**: 주의사항이나 제약조건을 포함할 수 있습니다.
3. **3단계**: 성공적인 수행을 위한 구체적인 명령어나 과정을 나열합니다.

- `scripts/`: 이 스킬과 관련된 자동화 스크립트나 도구들을 보관합니다.
- `examples/`: 참고할 수 있는 예시 파일, 구현 패턴들을 보관합니다.
- `resources/`: 추가적인 텍스트 리소스나 설정 파일들을 보관합니다.
